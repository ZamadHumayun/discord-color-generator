How to write colored text in Discord?

What is Discord ANSI formatting?

Discord uses Markdown formatting for message styles. By wrapping text inside an `ansi` code block, you can use ANSI escape sequences (control codes) to color text and apply bold/underline styles without any bots or plugins.

Does colored text work on Discord Mobile?

Currently, Discord ANSI colors only render inside code blocks on the Discord Desktop client and the browser-based Web App. On Discord Mobile (iOS/Android), the text will show up as standard white/gray code block text.

How do I use this generated code?

Simply click the "Copy Code" button, head over to your Discord app, paste the copied text into any text channel, and press enter. It will automatically render the formatted code block.

Why is my text showing strange characters?

Make sure you have copied the entire code block including the triple backticks (```ansi) at the start and the triple backticks (```) at the end. The strange characters like [1;31m are the ANSI codes that Discord needs to render the color.
